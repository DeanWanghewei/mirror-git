系统信息
自动同步

系统会根据配置的时间间隔自动同步所有启用的仓库。

同步间隔：

  • 默认间隔：每 1 小时（3600秒）
  • 可通过环境变量 SYNC_INTERVAL 自定义
时区设置

所有时间将自动转换为您浏览器的本地时区显示。

可通过环境变量 TIMEZONE 设置应用时区（默认：Asia/Shanghai）

Gitea 配置说明
配置方式

Gitea 相关配置只能通过环境变量设置，无法在 Web UI 中修改。

环境变量：

  • GITEA_URL - Gitea 服务器地址
  • GITEA_TOKEN - Gitea 访问令牌
  • GITEA_USERNAME - Gitea 用户名

详细配置请查看 README.md
